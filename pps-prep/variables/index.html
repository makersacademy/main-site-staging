Learn Ruby Online

Lesson 13: Variables

LEARNING OUTCOMES
Understand what a variable is
Explain what the assignment operator does
How to define a variable in Ruby
Recognise that a new value will overwrite the contents of your variable
Be able to retrieve a value by calling the variable you stored it in
EXERCISES
1. Create a variable
2. Assign a value to that variable
3. Retrieve that value by calling the variable's name (in the computer - not verbally)
4. Assign another value to the same variable
5. Hypothesize about what you will see now when you call the variable again
6. Repeat
